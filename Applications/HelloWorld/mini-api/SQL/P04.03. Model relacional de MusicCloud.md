# Inserir àlbums

Com que `Albums` necessita `artistId`, primer hem de recuperar els identificadors dels artistes.

Podem inserir els àlbums utilitzant subconsultes:

```sql
INSERT INTO Albums (title, artistId, releaseYear)
VALUES
  (
    'A Night at the Opera',
    (SELECT id FROM Artists WHERE name = 'Queen'),
    1975
  ),
  (
    'Thriller',
    (SELECT id FROM Artists WHERE name = 'Michael Jackson'),
    1982
  ),
  (
    'Discovery',
    (SELECT id FROM Artists WHERE name = 'Daft Punk'),
    2001
  ),
  (
    'Nevermind',
    (SELECT id FROM Artists WHERE name = 'Nirvana'),
    1991
  );
GO
```

Comprovem:

```sql
SELECT 
  Albums.id,
  Albums.title,
  Artists.name AS artist,
  Albums.releaseYear
FROM Albums
INNER JOIN Artists ON Albums.artistId = Artists.id;
GO
```

# Consultar tracks amb artista i àlbum

```sql
SELECT 
  Tracks.id,
  Tracks.title,
  Artists.name AS artist,
  Albums.title AS album,
  Tracks.durationSeconds
FROM Tracks
INNER JOIN Artists ON Tracks.artistId = Artists.id
LEFT JOIN Albums ON Tracks.albumId = Albums.id;
GO
```

Fem servir:

```text
INNER JOIN Artists
```

perquè tota cançó ha de tenir artista.

Fem servir:

```text
LEFT JOIN Albums
```

perquè una cançó pot no tenir àlbum.

# Crear playlists

```sql
INSERT INTO Playlists (name, description, userId)
VALUES
  (
    'Clàssics imprescindibles',
    'Cançons conegudes de diferents èpoques',
    (SELECT id FROM Users WHERE username = 'anna')
  ),
  (
    'Música per programar',
    'Tracks per concentrar-se mentre escrivim codi',
    (SELECT id FROM Users WHERE username = 'marc')
  );
GO
```

Comprovem:

```sql
SELECT 
  Playlists.id,
  Playlists.name,
  Playlists.description,
  Users.username AS owner
FROM Playlists
INNER JOIN Users ON Playlists.userId = Users.id;
GO
```

# Consultar el contingut d'una playlist

```sql
SELECT 
  Playlists.name AS playlist,
  PlaylistTracks.position,
  Tracks.title,
  Artists.name AS artist,
  Tracks.durationSeconds
FROM PlaylistTracks
INNER JOIN Playlists ON PlaylistTracks.playlistId = Playlists.id
INNER JOIN Tracks ON PlaylistTracks.trackId = Tracks.id
INNER JOIN Artists ON Tracks.artistId = Artists.id
WHERE Playlists.name = 'Clàssics imprescindibles'
ORDER BY PlaylistTracks.position;
GO
```

Aquesta consulta mostra les cançons d'una playlist ordenades per posició.

Aquest tipus de consulta serà molt útil més endavant per a un endpoint com:

```text
GET /playlists/:id/tracks
```

# Comprovar que una cançó pot estar en més d'una playlist

Afegim `Bohemian Rhapsody` també a la playlist **Música per programar**:

```sql
INSERT INTO PlaylistTracks (playlistId, trackId, position)
VALUES
  (
    (SELECT id FROM Playlists WHERE name = 'Música per programar'),
    (SELECT id FROM Tracks WHERE title = 'Bohemian Rhapsody'),
    3
  );
GO
```

Ara consultem totes les playlists on apareix aquesta cançó:

```sql
SELECT 
  Tracks.title,
  Playlists.name AS playlist,
  PlaylistTracks.position
FROM PlaylistTracks
INNER JOIN Tracks ON PlaylistTracks.trackId = Tracks.id
INNER JOIN Playlists ON PlaylistTracks.playlistId = Playlists.id
WHERE Tracks.title = 'Bohemian Rhapsody';
GO
```

Això demostra la relació molts-a-molts.
