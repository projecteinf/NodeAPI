# POST

## Obtenir ID artista

Per donar d'altra un track, aquest ha d'estar associat obligatòriament a un artista.


```bash
docker exec -it musiccloud-sqlserver /opt/mssql-tools18/bin/sqlcmd \
  -S localhost \
  -U sa \
  -P "Patata123!" \
  -No \
  -d MusicCloud \
  -Q "SELECT CONVERT(NVARCHAR(36), id) AS id, title FROM Albums;"
```

---
## Obtenir ID àlbum

L'àlbum és opcional. Per tant, podem obviar aquest camp.

```bash
docker exec -it musiccloud-sqlserver /opt/mssql-tools18/bin/sqlcmd \
  -S localhost \
  -U sa \
  -P "Patata123!" \
  -No \
  -d MusicCloud \
  -Q "SELECT CONVERT(NVARCHAR(36), id) AS id, title FROM Albums;"
```
