## Learning outcomes

By the end of this tutorial, students will be able to:

- Explain the role of routes, controllers, services and models.
- Build REST endpoints using Express and TypeScript.
- Validate request bodies, route parameters and query strings.
- Model relational data using MSSQL.
- Implement authentication with JWT.
- Protect endpoints using middleware.
- Return consistent HTTP status codes and error responses.
- Write basic integration tests for API endpoints.
- Document an API using OpenAPI/Swagger.
