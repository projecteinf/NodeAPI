# El meu primer Component amb React

**Objectiu:** Aprendre a crear una arquitectura bàsica de components amb React. Migrarem la col·lecció de dades que hem fet servir a les unitats anteriors cap a una interfície modular generada completament en segon pla per Babel i React.

# Preparació de la plantilla i l'entorn CDN

1. Crea un fitxer al teu ordinador anomenat `indexReact.html`.
2. Enganxa-hi l'estructura base que conté els estils de Spotify i l'array original de cançons:

```html
<!DOCTYPE html>
<html lang="ca">
	<head>
	    <meta charset="UTF-8">
	    <title>El meu primer React - Spotify</title>
	    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
	    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
	    <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
	    
	    <style>
	        body { background-color: #121212; color: white; font-family: sans-serif; padding: 20px; }
	        .playlist-title { color: #1DB954; }
	        .grid-cancons { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); gap: 15px; margin-top: 20px; }
	        .targeta-canco { background-color: #181818; padding: 15px; border-radius: 8px; border: 1px solid #282828; }
	        .targeta-canco h1 { margin: 0 0 10px 0; font-size: 16px; color: #1DB954; }
	        .targeta-canco p { margin: 5px 0; color: #b3b3b3; font-size: 14px; }
	    </style>
	</head>
	<body>
	
	    <div id="root"></div>
	
	    <script type="text/babel">
	
	        // 1. Les dades mestres (Simulació del nostre catàleg)
	        const baseDeDadesMusica = [
	            { id: 1, titol: "Despechá", artista: "Rosalía", genere: "Pop" },
	            { id: 2, titol: "Candy", artista: "Rosalía", genere: "Pop" },
	            { id: 3, titol: "Monaco", artista: "Bad Bunny", genere: "Trap" },
	            { id: 4, titol: "Ojitos Lindos", artista: "Bad Bunny", genere: "Reggaeton" }
	        ];
	
	        // LES TEVES TASQUES COMENÇEN AQUÍ
	
	    </script>
	</body>
</html>

```

# Selecció de la dada i arrencada del motor

Sota de les dades mestres, realitza les següents configuracions inicials:

1. Crea una constant anomenada `cancoFavorita` i assigna-li, mitjançant el seu índex, la **primera cançó** de l'array `baseDeDadesMusica`.
2. Captura el contenidor de l'HTML fent servir `document.querySelector('#root')`.
3. Inicialitza l'arrel de React-DOM executant `ReactDOM.createRoot()` sobre el contenidor capturat i guarda-ho en una constant anomenada `arrelReact`.

# Crear el component fill `<TargetaCanco />`

Anem a dissenyar la nostra primera peça del trencaclosques.

1. Declara un component de React (funció fletxa) anomenat `TargetaCanco`. Recorda la norma d'or: **els components han de començar sempre en majúscula**.
2. Fes que el component retorni directament un bloc de JSX amb l'estructura d'una targeta: un `<div>` amb la classe corresponent (`className="targeta-canco"`).
3. A dins del div, utilitza les claus `{}` per injectar de forma dinàmica el títol de la cançó dins d'un `<h1>` i el nom de l'artista dins d'un `<p>`, agfant-los de la constant `cancoFavorita` que has preparat a la Tasca 2.

# Crear el component pare `<App />` i renderitzar

A la vida real, una web de React es compon d'un component principal que agrupa els altres.

1. Crea un component anomenat `App`.
2. Aquest ha de retornar un `<div>` general que contingui un títol principal (`<h1 className="playlist-title">Cançons preferides</h1>`) i un contenidor de tipus graella (`<div className="grid-cancons">`).
3. Dins de la graella, crida el teu component fill de la Tasca 3 com si fos una etiqueta HTML: `<TargetaCanco />`.
4. Al final de tot del script, crida el mètode `.render()` de la teva `arrelReact` pasant-li el component pare: `<App />`.
5. Obre la pàgina al navegador i comprova que la teva primera cançó es dibuixa automàticament a la pantalla.
