# Pintant el Catàleg de Spotify

**Objectiu:** Utilitzar les funcions pures de la Unitat 2 i combinar-les amb la manipulació del DOM per agafar un array d'objectes (cançons) i pintar-los dinàmicament a la pantalla creant targetes HTML complexes.
# Preparació de l'entorn HTML/CSS

Aquesta vegada no treballarem sobre una pestanya buida (`about:blank`).
Crea un fitxer anomenat `index.html` al teu ordinador i enganxa-hi aquesta estructura base que simula la interfície de Spotify:
```html
<!DOCTYPE html>
<html lang="ca">
	<head>
	    <meta charset="UTF-8">
	    <title>Spotify de Classe - DOM</title>
	    <style>
	        body { background-color: #121212; color: white; font-family: sans-serif; padding: 20px; }
	        .playlist-title { color: #1DB954; } /* Verd Spotify */
	        .grid-cancons { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); gap: 15px; margin-top: 20px; }
	        .targeta-canco { background-color: #181818; padding: 15px; border-radius: 8px; border: 1px solid #282828; }
	        .targeta-canco h1 { margin: 0 0 10px 0; font-size: 16px; }
	        .targeta-canco p { margin: 5px 0; color: #b3b3b3; font-size: 14px; }
	    </style>
	</head>
	<body>
	
	    <h1 class="playlist-title">El teu Catàleg Musical</h1>
	    
	    <div id="contenedor-principal" class="grid-cancons"></div>
	
	    <script>
	        // Les dades mestres (Simulació del Backend)
	        const baseDeDadesMusica = [
	            { id: 1, titol: "Despechá", artista: "Rosalía", durada: 157, genere: "Pop" },
	            { id: 2, titol: "Candy", artista: "Rosalía", durada: 191, genere: "Pop" },
	            { id: 3, titol: "Monaco", artista: "Bad Bunny", durada: 207, genere: "Trap" },
	            { id: 4, titol: "Ojitos Lindos", artista: "Bad Bunny", durada: 265, genere: "Reggaeton" }
	        ];
	
	        // LES TEVES TASQUES VAN AQUÍ AVALL
	    </script>
	</body>
</html>
```
# Crear targeta cançó

Dins de l'etiqueta `<script>`, has de programar una funció arrow `crearTargetaHtml`.

1. Reb com a paràmetre un únic **objecte cançó** (per exemple, `{titol: "...", artista: "...", ...}`).
2. Crear un element a la memòria: un `<div>` amb una classe CSS `targeta-canco` que farà de contenidor de la targeta. 
3. Dins d'aquest `<div>`, ha de crear i introduir un `<h1>` amb el títol de la cançó i un `<p>` amb el nom de l'artista.
4. Finalment, la funció **ha de retornar** el `<div>` resultant.

# Pintar el catàleg sencer (Ús de `foreach`)

1. Crea una funció arrow anomenada `renderitzarCatatleg`.
2. Reb com a paràmetres la llista d'objectes a pintar (array) i el contenidor destí del HTML on s'han d'enganxar.
3. Aplica la funció  `crearTargetaHtml` a tots els elements de l'array.
4. Executa la funció pasant-li `baseDeDadesMusica` i el contenidor principal.
