# "El Motor del Catàleg de Spotify"

**Objectiu:** Programar la lògica interna del cercador, llistat i reproductor de cançons de l'aplicació mitjançant funcions i mètodes d'arrays.
### Tasca 1: Creació del Catàleg Base

Obre la consola del navegador (`about:blank` + `F12`) i defineix la teva base de dades musical executant aquest array d'objectes:

```javascript
const baseDeDadesMusica = [
    { id: 1, titol: "Despechá", artista: "Rosalía", durada: 157, genere: "Pop" },
    { id: 2, titol: "Candy", artista: "Rosalía", durada: 191, genere: "Pop" },
    { id: 3, titol: "Monaco", artista: "Bad Bunny", durada: 207, genere: "Trap" },
    { id: 4, titol: "Ojitos Lindos", artista: "Bad Bunny", durada: 265, genere: "Reggaeton" },
    { id: 5, titol: "Milionària", artista: "Rosalía", durada: 138, genere: "Rumba" }
];
```
### Tasca 2: Crear el Cercador per Artista (Ús de `.filter()`)

1. Programa una funció fletxa anomenada `buscarPerArtista` que retorni totes les cançons d'un artista.

2. Dins de la funció, utilitza el mètodes `.filter()` sobre `baseDeDadesMusica` per trobar només les cançons d'aquell artista. La funció ha de retornar aquest nou array filtrat.

3. Executa la funció a la consola passant-li `"Rosalía"` com a argument i comprova que et retorna un array amb 3 elements.

### Tasca 3: El Formatador de Durada 

A Spotify, el reproductor no mostra el temps en segons (ex: `157`), sinó en format minuts:segons (ex: `2:37`).

1. Crea una funció anomenada `formatarTemps` que rebi un número (segons).
2. Fent servir operacions matemàtiques bàsiques (pista: utilitza `Math.floor(segons / 60)` per als minuts i l'operador residu `%` per als segons restants), calcula els minuts i segons.
3. Fes que la funció retorni el text en format `MM:SS`.

### Tasca 4: Llistar la Playlist de l'Usuari (Ús de `.map()`)

Imagina que l'usuari obre un àlbum. Volem transformar la base de dades de cançons en línies de text estilitzades llestes per imprimir per pantalla.

1. Crea una funció anomenada `generarLlistaVisual`.
2. Dins de la funció, aplica un `.map()` a la `baseDeDadesMusica`. Per a cada cançó, ha de generar un text combinant el títol, l'artista i la durada ja formatada (crida la funció de la Tasca 3 dins del `.map()`).
3. El format del text resultant ha de ser: `"Títol - Artista (Minuts:Segons)"`.
4. Retorna el nou array de textos i executa la funció per comprovar-ho.

### Tasca 5: El mètode Debugger en bucles agregats

1. Afegeix un `debugger;` dins de la funció de la Tasca 4, just dins de la lògica del `.map()`.
2. Executa la funció. El codi es congelarà.
3. Utilitza les eines de pas de la depuració per veure com el navegador executa el codi "en bucle", passant per la cançó 1, després per la 2, etc., transformant-les una a una.

### Requisits del lliurament:

1. El codi de les 4 funcions programades.
2. Captura de pantalla del resultat de l'execució de `generarLlistaVisual()` on es vegi l'array final de cadenes de text formatades amb els minuts correctament calculats.
3. Breu explicació a la memòria de per què el mètode `.map()` és diferent del mètode `.filter()`.
