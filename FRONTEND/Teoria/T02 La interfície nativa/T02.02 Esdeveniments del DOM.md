# Esdeveniments del DOM, Escoltar el Client i Gestió de Formularis

## Què és un esdeveniment (*Event*)?

Un **esdeveniment** és qualsevol acció que l'usuari realitza sobre la pàgina web i que el navegador és capaç de detectar. Exemples comuns:

* Fer un clic amb el ratolí (`click`).
* Escriure en un camp de text (`input` o `change`).
* Enviar un formulari premut el botó de *Submit* (`submit`).

Per fer que el nostre codi reaccioni, hem d'enganxar un "micròfon" a l'etiqueta HTML que volem vigilar. En JavaScript, aquest micròfon es diu **`addEventListener()`**.

```javascript
const botoPlay = document.querySelector('.btn-play');

// Li diem: "Quan detectis un 'click', executa aquesta funció fletxa"
botoPlay.addEventListener('click', () => {
    console.log("L'usuari ha premut el botó de reproducció!");
});

```
## L'objecte Esdeveniment (`event` o `e`)

Cada vegada que s'activa un esdeveniment, JavaScript genera de forma automàtica un objecte amb tota la informació del que acaba de passar (quin botó s'ha clicat, quines tecles s'han premut, en quines coordenades està el ratolí, etc.).

Per rebre'l, només l'hem de passar com a paràmetre a la nostra funció:

```javascript
botoPlay.addEventListener('click', (e) => {
    // 'e.target' ens diu exactament quina etiqueta HTML ha rebut el clic
    console.log("Has fet clic a l'etiqueta: ", e.target);
});
```
## Gestió de Formularis i l'efecte per defecte

Quan volem que l'usuari introdueixi dades (com ara crear una nova cançó per al catàleg), utilitzem l'esdeveniment `'submit'` associat a l'etiqueta `<form>`.

Però els formularis tenen un comportament natiu perillós de l'època dels anys 90: quan fas clic a enviar, la pàgina web es **refresca sencera**. Si la pàgina es reinicia, perdem l'estat de la nostra aplicació (la música que estava sonant s'aturaria).

Per evitar-ho, la primera línia de qualsevol formulari modern en JavaScript ha de ser sempre **`e.preventDefault()`** (bloqueja el refresc automàtic).

```javascript
const formulari = document.querySelector('#formulari-canco');

formulari.addEventListener('submit', (e) => {
    e.preventDefault(); // ATURAT! El navegador ja no refrescarà la pàgina
    
    console.log("Formulari rebut de manera segura en segon pla.");
});
```
