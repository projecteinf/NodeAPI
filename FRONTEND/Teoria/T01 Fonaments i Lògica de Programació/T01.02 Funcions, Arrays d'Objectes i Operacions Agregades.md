# Funcions, Arrays d'Objectes i Operacions Agregades

## El concepte de Funció: Reutilitzar codi

Fins ara, hem escrit codi que s'executa un sol cop de dalt a baix. Però, què passa si volem validar un usuari o reproduir una cançó en diferents moments de la nostra aplicació? No anem a copiar i enganxar 20 línies de codi cada vegada.

Una **funció** és un bloc de codi amb un nom que conté una tasca específica i que podem "cridar" (executar) tants cops com vulguis.
### Funcions tradicionals vs. *Arrow Functions*

En JavaScript modern utilitzem molt les funcions fletxa per la seva sintaxi compacta (i perquè React les fa servir al 100% per crear components).

```javascript
// 1. Funció tradicional
function saludaUsuari(nom) {
    return "Hola, " + nom + "! Benvingut a Spotify.";
}

// 2. Funció Fletxa (Arrow Function) - Fan exactament el mateix
const saludaUsuariFletxa = (nom) => {
    return "Hola, " + nom + "! Benvingut a Spotify.";
};

// Com es criden?
console.log(saludaUsuari("Marta"))
console.log(saludaUsuariFletxa("Marta")); 
```
## Estructures de dades complexes: Objectes i Llistes (Arrays)

Per fer un clon de Spotify, una cançó no és només un text. Una cançó té un títol, un artista, una durada en segons i si està marcada com a "premium".

### Els Objectes (L'estructura d'una cançó)

Un objecte ens permet agrupar propietats (clau-valor) dins d'una sola variable utilitzant les claus `{}`:

```javascript
const canco = {
    titol: "Despechá",
    artista: "Rosalía",
    durada: 157, // en segons
    esPremium: false
};

// Accedir a les propietats (Sintaxi del punt)
console.log(canco.titol + " té una durada de "+canco.durada+" segons"); // Mostra: Despechá
```

#### B. El Catàleg: Arrays d'Objectes

Un **Array** (matriu o col·lecció) és una llista ordenada d'elements que va entre claudàtors `[]`. Si ajuntem els arrays i els objectes, podem crear el nostre catàleg musical o *Playlist*:
```javascript
const playlist = [
    { titol: "Despechá", artista: "Rosalía", durada: 157 },
    { titol: "Candy", artista: "Rosalía", durada: 191 },
    { titol: "Monaco", artista: "Bad Bunny", durada: 207 }
];
```
## Operacions Agregades Modernes (`.filter()` i `.map()`)

Quan tinguem milers de cançons a la base de dades, necessitarem filtrar-les o transformar-les. JavaScript té mètodes natius que recorren les llistes de forma automàtica.
### `.filter():` Filtrar dades

Crea una **nova llista** que només conté els elements que compleixen una condició. Reben una funció com a paràmetre:

```javascript
const filtrarPerRosalia = (llistatCancons) => {
 return llistatCancons.filter(canco => canco.artista === "Rosalía");
}
// 'canconsRosalia' ara conté un array amb només les dues primeres cançons.
console.log(filtrarPerRosalia(playlist));
```

### `.map():` Transformar dades amb 

Recorre l'array original i **transforma cada element** en una altra cosa, tornant un array nou de la mateixa llargada. *Això serà el que farà servir React per convertir objectes JavaScript en targetes visuals de HTML.*

```javascript
// Volem una llista que només tingui els títols en majúscules
const obtenirTitolsMajuscules = (cancons) => {
 return cancons.map(canco => canco.titol.toUpperCase());
}
// Resultat: ["DESPECHÁ", "CANDY", "MONACO"]
console.log(obtenirTitolsMajuscules(playlist))
```
